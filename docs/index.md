# Template Sample

A comprehensive test project with all template features enabled

## Quick Start

```bash
# Install the package
pip install template-sample

# Or install with development dependencies
uv sync --all-extras
```

## CLI Usage

```bash
# Show help
template_sample --help

# Example command
template_sample hello --name "World"
```
## Features

- Modern Python 3.12+ support
- Type-safe with BasedPyright strict mode
- Comprehensive test coverage
- Structured logging with structlog
- CLI interface with Click
- Docker support
